ngView

Директива позволяет добавлять анимационные эффекты на появление/исчезание элементов, у которых есть указана директива ng-view

Анимации
.animate .animate-enter { -webkit-transition: 1s linear all; /* Chrome */ transition: 1s linear all; opacity: 0; } .animate .animate-enter.animate-enter-active { opacity: 1; }
